## 느낀점 & 개선할 점

``style class`` 의 이름에 ``-`` 를 단위로 ``하위 요소`` 를 구분하고 싶었습니다.

``요소의 깊이`` 가 깊어질수록 ``style class`` 명이 길어지는 현상이 발생 하였습니다.

다음 퍼블리싱에는 ``최상위 요소`` 의 ``style class`` 명을 고정적으로 참조하는 방식으로 ``style class 명``을 작성해볼 계획 입니다.
